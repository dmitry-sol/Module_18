Установка Django:
pip Install django

Создание проекта "UrbanDjango":
django-admin startproject UrbanDjango

Создание таблицы в базе данных для всех приложений из списка INSTALLED_APPS
cd UrbanDjango
python manage.py migrate

Создание приложения:
python manage.py startapp example1
python manage.py startapp task2


- - - - -
Инициирующая миграция и последующие миграции после изменения models:
python manage.py makemigrations task2

Вывести в консоль SQL первой миграции:
python manage.py sqlmigrate task2 0001

Применение миграции:
python manage.py migrate
- - - - -


Для управление сайтом, создание аккаунт суперпользователя:
python manage.py createsuperuser

Добавление модели task2 на сайт администрирования:

from django.contrib import admin 
from .models import Post 

admin.site.register(Post)
